# 云端 AI 病虫害识别配置指南

## 📋 功能说明

病虫害诊断页面已集成**云端大模型**，每次上传图片后会自动调用视觉 API 进行智能识别。

## 🔑 配置步骤

### 方法1：使用环境变量（推荐）

1. **获取 API 密钥**
   - 访问：https://www.volcengine.com/
   - 注册/登录账号
   - 进入控制台，创建 API 密钥

2. **设置环境变量**
   ```bash
   # Windows PowerShell
   $env:DOUBAO_API_KEY="你的API密钥"
   
   # Windows CMD
   set DOUBAO_API_KEY=你的API密钥
   
   # Linux/Mac
   export DOUBAO_API_KEY="你的API密钥"
   ```

3. **启动服务**
   ```bash
   python app.py
   ```

### 方法2：修改配置文件

1. **编辑配置文件**
   - 打开 `doubao_config.py`
   - 找到 `DOUBAO_API_KEY` 变量
   - 将 `''` 改为你的 API 密钥：`DOUBAO_API_KEY = '你的API密钥'`

2. **保存并启动**
   ```bash
   python app.py
   ```

## 🚀 使用方法

1. 访问病虫害诊断页面：`http://127.0.0.1:5000/病虫害`
2. 点击上传区域或拖拽图片
3. 点击"开始AI诊断"按钮
4. 等待云端分析（通常需要5-15秒）
5. 查看详细的诊断报告

## 📝 诊断报告格式

云端模型会提供以下信息：
- **病虫害类型识别**：具体识别出的病虫害名称
- **严重程度评估**：轻度/中度/重度及判断依据
- **防治建议**：具体的防治措施和用药建议

## ⚠️ 注意事项

1. **API配额**：请确保您的 API 账户有足够的调用配额
2. **网络连接**：需要稳定的网络连接才能调用 API
3. **图片格式**：支持 JPG、PNG、GIF、WEBP 格式
4. **图片大小**：建议不超过16MB

## 🔧 故障排查

### 问题1：提示"API密钥未配置"
- **解决**：按照上述配置步骤设置 API 密钥

### 问题2：提示"API调用失败"
- **检查**：API 密钥是否正确
- **检查**：网络连接是否正常
- **检查**：API 配额是否充足

### 问题3：响应超时
- **解决**：稍后重试，或检查网络连接

## 📞 技术支持

如有问题，请检查：
1. `doubao_config.py` 配置文件
2. `app.py` 中的 `call_doubao_api` 函数
3. 官方文档：https://www.volcengine.com/docs/82379
