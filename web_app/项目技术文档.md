## Demo1 - 苹果成熟度检测模型

### 1.1 任务描述

**目标**: 识别苹果的成熟度状态，分为3个类别：
- **未成熟**: 呈绿色，尚未成熟
- **成熟**: 呈红色（包括半成熟和完全成熟）
- **过成熟**: 可能已腐烂，需要及时处理

### 1.2 模型架构

#### 1.2.1 基础网络
- **骨干网络**: ResNet50（ImageNet预训练权重）
- **输入尺寸**: 224×224×3
- **输出类别数**: 3

#### 1.2.2 注意力机制

模型采用 **CBAM（Convolutional Block Attention Module）** 注意力机制，在ResNet的关键层后添加注意力模块：

**CBAM结构**:
```
CBAM = Channel Attention + Spatial Attention
```

**通道注意力（Channel Attention）**:
- 使用全局平均池化和全局最大池化
- 通过两个全连接层学习通道权重
- 压缩比（ratio）: 16

**空间注意力（Spatial Attention）**:
- 在通道维度上计算平均值和最大值
- 通过卷积层学习空间权重
- 卷积核大小: 7×7

**注意力模块位置**:
- `layer2` 后（512通道）
- `layer3` 后（1024通道）
- `layer4` 后（2048通道）

#### 1.2.3 分类头设计

```python
分类头结构:
- Dropout(0.5)
- Linear(2048 → 512)
- ReLU
- Dropout(0.3)
- Linear(512 → 3)
```

### 1.3 数据预处理

#### 1.3.1 数据清洗
- 检查并删除损坏的图片
- 统一图片格式为RGB
- 统一图片尺寸为224×224

#### 1.3.2 数据集划分
- **训练集**: 70%
- **验证集**: 15%
- **测试集**: 15%

#### 1.3.3 数据增强策略（训练集）

**基础增强**:
- `RandomResizedCrop(224, scale=(0.7, 1.0))`: 随机裁剪并调整大小
- `RandomRotation(30)`: 随机旋转±30度
- `RandomHorizontalFlip()`: 随机水平翻转
- `RandomVerticalFlip()`: 随机垂直翻转

**颜色增强**:
- `ColorJitter(brightness=0.4, contrast=0.4, saturation=0.4, hue=0.15)`: 颜色抖动，增强模型对颜色变化的鲁棒性

**几何变换**:
- `RandomAffine(degrees=10, translate=(0.15, 0.15), scale=(0.9, 1.1))`: 随机仿射变换
- `RandomPerspective(distortion_scale=0.2, p=0.3)`: 随机透视变换

**正则化增强**:
- `RandomErasing(p=0.3, scale=(0.02, 0.4))`: 随机擦除，防止过拟合

**验证/测试集预处理**:
- `Resize(256)`
- `CenterCrop(224)`
- `Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])`

### 1.4 训练策略

#### 1.4.1 损失函数
- **Focal Loss**: 处理类别不平衡问题
  - `alpha = 1`
  - `gamma = 2`
  - 专注于难分类样本，降低易分类样本的权重

#### 1.4.2 优化器
- **优化器**: SGD with Momentum
- **学习率**: 0.0001（初始）
- **动量**: 0.9
- **权重衰减**: 0.0001（L2正则化）

#### 1.4.3 学习率调度
- **策略**: Warmup + Cosine Annealing
- **Warmup阶段**: 前3个epoch线性增加学习率
- **Cosine阶段**: 后续epoch按余弦函数衰减

#### 1.4.4 训练流程

**分阶段训练**:
1. **第一阶段（冻结阶段）**: 前10个epoch
   - 冻结ResNet骨干网络参数
   - 只训练分类头和注意力模块
   - 学习率: 0.001（10倍初始学习率）

2. **第二阶段（微调阶段）**: 后续epoch
   - 解冻所有参数
   - 微调整个网络
   - 学习率: 0.0001

#### 1.4.5 正则化技术
- **Dropout**: 0.5（分类头第一层）、0.3（分类头第二层）
- **梯度裁剪**: `max_norm=1.0`，防止梯度爆炸
- **权重衰减**: 0.0001

#### 1.4.6 早停机制
- **耐心值**: 10个epoch
- **监控指标**: 验证集准确率
- **保存策略**: 保存验证集准确率最高的模型

#### 1.4.7 测试时增强（TTA）
- 对测试图片进行水平翻转
- 对原始预测和翻转预测取平均
- 提升模型鲁棒性

### 1.5 超参数配置

| 参数 | 值 |
|------|-----|
| Batch Size | 16 |
| Epochs | 50 |
| 初始学习率 | 0.0001 |
| 图像尺寸 | 224×224 |
| 早停耐心值 | 10 |
| 冻结阶段epochs | 10 |
| 微调阶段epochs | 40 |

## Demo2 - 多类别水果分类模型

### 2.1 任务描述

**目标**: 识别5种不同的水果类别：
- **Apple**: 苹果
- **Carambola**: 杨桃
- **Pear**: 梨
- **Plum**: 李子
- **Tomatoes**: 番茄

### 2.2 模型架构

#### 2.2.1 基础网络
- **骨干网络**: ResNet50（ImageNet预训练权重）
- **输入尺寸**: 224×224×3
- **输出类别数**: 5

#### 2.2.2 增强注意力机制

模型采用 **增强CBAM + SE注意力** 机制，相比Demo1更加复杂：

**增强通道注意力（Enhanced Channel Attention）**:
- 添加BatchNorm层提升稳定性
- 压缩比（ratio）: 8（比Demo1更小，增强注意力）
- 使用BatchNorm2d进行归一化

**增强空间注意力（Enhanced Spatial Attention）**:
- 添加BatchNorm层
- 卷积核大小: 7×7

**SE注意力（Squeeze-and-Excitation）**:
- 作为CBAM的补充
- 通过全局平均池化和全连接层学习通道权重
- 压缩比: 8

**注意力模块位置**:
- `layer1` 后（256通道）- **新增**
- `layer2` 后（512通道）
- `layer3` 后（1024通道）
- `layer4` 后（2048通道）
- `final_attention`（2048通道）- **新增，在池化前再次应用注意力**

#### 2.2.3 分类头设计

```python
分类头结构:
- Dropout(0.5)
- Linear(2048 → 1024)
- BatchNorm1d(1024)
- ReLU
- Dropout(0.4)
- Linear(1024 → 512)
- BatchNorm1d(512)
- ReLU
- Dropout(0.3)
- Linear(512 → 5)
```

**特点**:
- 更深的分类头（3层全连接）
- 每层都添加BatchNorm1d
- 渐进式Dropout（0.5 → 0.4 → 0.3）

### 2.3 数据预处理

#### 2.3.1 数据清洗
- 检查并删除损坏的图片
- 统一图片格式为RGB
- 统一图片尺寸为224×224

#### 2.3.2 数据集划分
- **训练集**: 70%
- **验证集**: 15%
- **测试集**: 15%

#### 2.3.3 数据增强策略（训练集）

**基础增强**:
- `Resize(256)`: 先调整到256
- `RandomResizedCrop(224, scale=(0.5, 1.0), ratio=(0.75, 1.33))`: 更大的随机裁剪范围
- `RandomRotation(45)`: 随机旋转±45度
- `RandomHorizontalFlip()`: 随机水平翻转
- `RandomVerticalFlip()`: 随机垂直翻转

**颜色增强**（特别针对Apple和Tomatoes的区分）:
- `ColorJitter(brightness=0.5, contrast=0.5, saturation=0.5, hue=0.2)`: 更强的颜色变化

**几何变换**:
- `RandomAffine(degrees=0, translate=(0.1, 0.1), scale=(0.9, 1.1))`: 随机仿射变换

**正则化增强**:
- `RandomErasing(p=0.3, scale=(0.02, 0.33), ratio=(0.3, 3.3))`: 随机擦除

**验证/测试集预处理**:
- `Resize(256)`
- `CenterCrop(224)`
- `Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])`

### 2.4 训练策略

#### 2.4.1 损失函数
- **Focal Loss**: 处理类别不平衡和难样本
  - `gamma = 2.0`
  - 使用类别权重（class_weights）
  - 特别增强Apple类别的权重（如果样本数不足）

#### 2.4.2 类别权重计算

**权重公式**:
```python
base_weight = (max_count / count) ** 1.5
```

**Apple类别特殊处理**:
- 如果Apple样本数少于平均值的80%，额外增强50%权重
- 公式: `weight = base_weight * 1.5`

#### 2.4.3 加权采样
- 使用 `WeightedRandomSampler` 平衡训练数据
- 根据类别权重进行采样
- 确保每个batch中各类别样本分布更均匀

#### 2.4.4 优化器
- **优化器**: Adam
- **学习率**: 0.0005
- **权重衰减**: 0.0001

#### 2.4.5 学习率调度
- **策略**: ReduceLROnPlateau
- **监控指标**: 验证集损失
- **衰减因子**: 0.5
- **耐心值**: 3个epoch

#### 2.4.6 混合精度训练（AMP）
- **启用**: FP16混合精度训练
- **优势**: 
  - 训练速度提升约2倍
  - 显存占用减少约50%
  - 使用 `GradScaler` 防止梯度下溢

#### 2.4.7 正则化技术
- **Dropout**: 0.5 → 0.4 → 0.3（渐进式）
- **BatchNorm**: 在分类头的每一层都添加
- **权重衰减**: 0.0001

#### 2.4.8 早停机制
- **耐心值**: 5个epoch
- **监控指标**: 验证集准确率
- **保存策略**: 保存验证集准确率最高的模型

### 2.5 超参数配置

| 参数 | 值 |
|------|-----|
| Batch Size | 64（自动降级到32如果内存不足） |
| Epochs | 20 |
| 初始学习率 | 0.0005 |
| 图像尺寸 | 224×224 |
| 早停耐心值 | 5 |
| 混合精度训练 | 启用（FP16） |
| 数据加载进程数 | 0（Windows单进程） |

## 技术对比总结

### 3.1 模型架构对比

| 特性 | Demo1 | Demo2 |
|------|-------|-------|
| 骨干网络 | ResNet50 | ResNet50 |
| 注意力机制 | CBAM | 增强CBAM + SE |
| 注意力位置 | layer2, layer3, layer4 | layer1, layer2, layer3, layer4, final |
| 压缩比 | 16 | 8（更小，增强注意力） |
| 分类头深度 | 2层 | 3层 |
| BatchNorm | 无 | 有（分类头每层） |

### 3.2 训练策略对比

| 特性 | Demo1 | Demo2 |
|------|-------|-------|
| 损失函数 | Focal Loss | Focal Loss + 类别权重 |
| 优化器 | SGD + Momentum | Adam |
| 学习率调度 | Warmup + Cosine | ReduceLROnPlateau |
| 训练方式 | 分阶段（冻结+微调） | 端到端训练 |
| 混合精度 | 否 | 是（FP16） |
| 加权采样 | 否 | 是 |
| 类别权重 | 无 | 有（特别增强Apple） |

### 3.3 数据增强对比

| 增强方法 | Demo1 | Demo2 |
|---------|-------|-------|
| 随机裁剪范围 | (0.7, 1.0) | (0.5, 1.0) |
| 旋转角度 | ±30° | ±45° |
| 颜色抖动强度 | 0.4 | 0.5 |
| 随机透视变换 | 有 | 无 |
| 随机擦除概率 | 0.3 | 0.3 |

### 3.4 核心技术要点

#### 3.4.1 注意力机制
- **Demo1**: 标准CBAM，在3个关键层添加
- **Demo2**: 增强CBAM + SE，在5个位置添加，压缩比更小，注意力更强

#### 3.4.2 类别不平衡处理
- **Demo1**: Focal Loss
- **Demo2**: Focal Loss + 类别权重 + 加权采样（三重保障）

#### 3.4.3 训练效率
- **Demo1**: 分阶段训练，先冻结后微调
- **Demo2**: 混合精度训练，速度提升2倍

#### 3.4.4 正则化
- **Demo1**: Dropout + 梯度裁剪
- **Demo2**: 渐进式Dropout + BatchNorm + 权重衰减

## 总结

1. **Demo1（苹果成熟度检测）**: 
   - 相对简单的3分类任务

   - 目标：平衡各类别准确率

两个模型都采用了：
- ✅ ImageNet预训练权重（迁移学习）
- ✅ 注意力机制（提升特征提取能力）
- ✅ 数据增强（提升泛化能力）
- ✅ Focal Loss（处理类别不平衡）
- ✅ 早停机制（防止过拟合）
